# simple-admin-panel

## Admin panel for managing companies and clients

### Test project (junior PHP developer)

Installation:

1. Clone repo

2. Create  database and add to .env

3. run ``` $ make install```

4. ```$ php artisan migrate --seed```

5. local start ```$ make start-app```
